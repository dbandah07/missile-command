# Missile Command (Unity Lab, C#)
Mechanics-focused Unity lab project inspired by the classic *Missile Command* arcade game. Built to practice targeting, projectile spawning, and collision-based gameplay loops.

## Features
- Mouse-based targeting and projectile firing
- Enemy spawn system with increasing pressure
- Collision detection and hit feedback
- Controls:
  - Desktop: Mouse aim + click to fire

## Notes
Completed during Fall 2025 as part of the class **Mobile Game Development**
